参数列表

参数名 类型 描述 默认值
width Int 宽度 120
top Int top 0
left Int left 0
items Array items null
shadow Bool 阴影 true
cls String css类名 null

Items

参数名 类型 描述 默认值
id   ID  
text   显示名  
icon   图标(liger css)  
img   图片路径  
disable   是否禁用  
children   子项  
click Function 点击事件  
dblclick Function 双击事件
