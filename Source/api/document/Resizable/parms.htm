参数列表

参数名 类型 描述 默认值
handles string 调整的位置 'n, e, s, w, ne, se, sw, nw'
maxWidth Int 最大宽度 2000
maxHeight Int 最多高度 2000
minWidth Int 最小宽度 20
minHeight Int 最小高度 20
scope Int scope 3
animate Bool 动画效果 false
