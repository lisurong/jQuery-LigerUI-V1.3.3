ligerComboBox

复杂的下拉框组件，适应不同类型的下拉框，包括：

  • 表单select元素
  • 单选、多选下拉框
  • 多列下拉框
  • 分页表格下拉框(包括搜索)
  • 树下拉框
  • 自动完成

示例

 1 var columns = [
 2                 { header: 'ID', name: 'id', width: 80 },
 3                 { header: '名字', name: 'name', width: 170 },
 4                 { header: '描述', name: 'desc', width: 170 }
 5             ];
 6 
 7             $("#txt1").ligerComboBox(
 8                 {
 9                     url: '../../data/net/ComboBoxData.ashx',
10                     valueField : 'id',
11                     textField: 'name',
12                     columns: columns,
13                     selectBoxWidth: 400,
14                     autocomplete: true,
15                     width: 400
16                 }
17             );
