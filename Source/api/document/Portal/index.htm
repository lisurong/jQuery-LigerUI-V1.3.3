ligerPortal

主要功能：

  1. 允许以多行、多列的方式自由得组合出Portal(基于Panel)
  2. 支持各种API方法：新增或删除Panel、加载远程数据等
  3. 支持加载远程数据加载Portal
  4. 支持记忆模式

示例

 2                 draggable : true,
 3                 columns: [{
 4                     width: 400,
 5                     panels: [{

10                         showClose: true
11                     },
12                     {

16                         content: '内容。。。'
