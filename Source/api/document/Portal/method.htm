方法列表

方法 参数 描述
remove(e)

e:rowIndex、columnIndex、index

rowIndex:行序号

columnIndex:列序号

index:Panel序号

 移出Panel
add(e)

e:rowIndex、columnIndex、index、panel

rowIndex:行序号

columnIndex:列序号

index:panel序号

panel:同panel参数

增加panel
getPanels() 数据结构： rowIndex: rowIndex,
                      columnIndex: columnIndex,
                      index: index,
                      panel : ligerUI Panel Object,
                      jpanel : jQuery panel Object 		获取Panels
getPanel(e)

e:rowIndex、columnIndex、index

rowIndex:行序号

columnIndex:列序号

index:Panel序号

 获取指定位置Panel
collapseAll()   收缩全部
expandAll()   展开全部
