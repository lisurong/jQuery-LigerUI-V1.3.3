参数列表

参数名 类型 描述 默认值
handler String handler null
proxy Object 代理 拖动时的主体,可以是'clone'或者是函数,返回jQuery 对象 true
revert Bool revert false
animate Bool 动画效果 true
receive Bool receive false
disabled Bool disabled false
proxyX Int 代理相对鼠标指针的位置,如果不设置则对应target的left null
proxyY Int 代理相对鼠标指针的位置,如果不设置则对应target的top null
clickDelay Int 鼠标按下再弹起，如果中间的间隔小于[dragDelay]毫秒，那么认为是点击，不会进行拖拽操作 100
minIsHide Bool 最小化仅隐藏
