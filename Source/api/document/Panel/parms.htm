参数列表

参数名 类型 描述 默认值
height Int 高度 300
width Int 宽度 400
title String 标题 'Panel'
content String 内容 null
url String 远程内容Url null
frameName String 创建iframe时 作为iframe的name和id null
data Object 可用于传递到iframe的数据 null
showClose Bool 是否显示关闭按钮 null
showToggle Bool 是否显示收缩按钮 null
icon String 左侧按钮 null
urlParms String url传承(支持函数) null
showRefresh Bool 显示刷新按钮 false
