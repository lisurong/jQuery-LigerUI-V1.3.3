事件列表

事件名 参数 描述
onBeforeExpand ({data,target}) 展开前事件
onContextmenu ({data,target}) 右击事件
onExpand ({data,target}) 展开事件
onBeforeCollapse ({data,target}) 收缩前事件
onCollapse ({data,target}) 收缩事件
onBeforeSelect ({data,target}) 选择前事件
onSelect ({data,target}) 选择事件
onBeforeCancelSelect ({data,target}) 取消选择前事件
onCancelselect ({data,target}) 取消选择事件
onCheck ({data,target},checked) 选择事件
onSuccess (data) 加载成功事件
onError (XMLHttpRequest, textStatus, errorThrown) 加载错误事件
onClick ({data,target}) 点击事件
onBeforeAppend (parentNode, newdata) 追加数据前事件
onAppend (parentNode, newdata) 追加数据事件
onAfterAppend (parentNode, newdata) 追加数据后事件
