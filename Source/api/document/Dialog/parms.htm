参数列表

参数名 类型 描述 默认值
cls String 给dialog附加css class null
id String 给dialog附加id null
buttons Array 按钮集合 null
isDrag Bool 是否拖动 true
width Int 宽度 280
height Int 高度 null
content String 内容 ''
target Object 目标对象，指定它将以appendTo()的方式载入 null
url String 标页url，默认以iframe的方式载入目 null
load String 是否以load()的方式加载目标页的内容 false
type String 类型 warn、success、error、question 'none'
left Int 相对窗口left null
top Int 相对窗口top null
modal Bool 是否模态对话框 true
name String 创建iframe时 作为iframe的name和id null
isResize Bool 是否调整大小 false
allowClose Bool 允许关闭 false
opener Object 引用opener到dialog对象 null
timeParmName String 是否给URL后面加上值为new Date().getTime()的参数，如果需要指定一个参数名即可 null
closeWhenEnter Bool 回车时是否关闭dialog null
isHidden Bool 关闭对话框时是否只是隐藏，还是销毁对话框 null
show Bool 初始化时是否马上显示 true
title String 标题 '提示'
showMax Bool 是否显示最大化 false
showToggle Bool 是否显示收缩 false
showMin Bool 是否显示最小化 false
slide Bool 动画效果 $.browser.msie ? false : true
fixedType String 在固定的位置显示, 可以设置的值有n, e, s, w, ne, se, sw, nw null
showType String 显示类型,可以设置为slide(固定显示时有效) null
contentCls String 设置内层div的样式 null
urlParms Object url参数 null
layoutMode Int 布局方式 1 九宫布局, 2 上中下布局 null
